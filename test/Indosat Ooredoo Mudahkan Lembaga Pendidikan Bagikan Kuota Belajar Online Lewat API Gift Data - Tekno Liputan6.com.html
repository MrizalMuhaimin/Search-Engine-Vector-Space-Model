    Indosat Ooredoo Mudahkan Lembaga Pendidikan Bagikan Kuota Belajar Online Lewat API Gift Data

    stella maris

    03 Nov 2020, 13:16 WIBDiperbarui 03 Nov 2020, 13:53 WIB

    11

    Liputan6.com, Jakarta Saat ini dunia pendidikan tengah beradaptasi dengan kebiasaan baru, yaitu melakukan Pembelajaran Jarak Jauh (PJJ) dengan mengandalkan teknologi internet. Agar para pelajar tetap semangat belajar, Indosat Ooredoo melalui Indosat Business menghadirkan Application Programming Interface (API) Gift Data.

    Ya, Indosat Ooredoo menyadari bahwa teknologi komunikasi kini menjadi hal paling esensial di tengah pandemi, untuk mengakomodasi kegiatan secara daring. Dalam rangka mendukung Pemerintah mengakomodir kebutuhan PJJ dan rencana bantuan kuota internet, API Gift Data pun menjadi solusi teknologi yang tepat bagi para institusi untuk dapat membagikan kuota internet ke peserta didik dan pengajar secara masif dan simple.

    Lalu apa itu API Gift Data yang disiapkan Indosat Ooredoo? Ini merupakan perangkat lunak yang memungkinkan pelanggan membagikan kuota internet ke nomor-nomor IM3 Ooredoo melalui sebuah dashboard.

    View this post on Instagram

    Sebagai bagian dalam mendukung program pembelajaran jarak jauh, Indosat Ooredoo menyediakan Platform API (Aplication Programming Interface) Gift Data yang memungkinkan pihak insitusi pendidikan mendistribusikan kuota data secara massive, seamless dan fleksibel. Pelajari lebih lanjut bagaimana kami memberikan solusi Pembelajaran Jarak Jauh di http://bit.ly/GiftData #TeknologiItuMemudahkan #TangguhBersamaIndonesia

    A post shared by indosatooredoobusiness (@indosatooredoobusiness) on Sep 18, 2020 at 5:25am PDT

    Chief Business Officer Indosat Ooredoo, Bayu Hanantasena, mengatakan bahwa Indosat Ooredoo terus berkomitmen mendukung Pemerintah Republik Indonesia untuk memasuki era adaptasi kebiasaan baru, khususnya untuk mendukung program Pembelajaran Jarak Jauh. "API Gift Data Indosat Ooredoo dapat mengirimkan kuota data secara massive, seamless, dan fleksibel," ujar Bayu.

    "API ini juga mendukung fitur automatisasi dan kustomisasi, yang memudahkan pengelolaan pengiriman kuota data sesuai kebutuhan setiap bulannya. Kami yakin platform ini akan efektif digunakan untuk penyaluran kuota internet kepada siswa-siswi serta guru agar pendidikan di Indonesia dapat terus berjalan," tambahnya

    2 dari 2 halaman

    Kontrol Jaringan Maksimal

    Tak hanya itu saja, Indosat Ooredoo juga menjaga kapasitas jaringan agar peserta didik dan para guru, agar dapat menjalankan proses pembelajaran dengan baik dan optimal.

    Indosat Ooredoo Network Operating Center (INOC) dan Indosat Ooredoo Service Operating Center (ISOC) melakukan monitoring dan kontrol jaringan selama 24x7 secara realtime, untuk memastikan pelanggan merasakan pengalaman digital terbaik.

    Manfaat INOC dan ISOC adalah untuk meningkatkan percepatan penyelesaian kendala sekaligus menjadi pusat koordinasi atau command center secara nasional. Untuk diketahui, pada Semester I Tahun 2020 Indosat Ooredoo mengoperasikan 124.944 BTS, dimana 52.776 merupakan BTS 4G. Penambahan 27.902 BTS 4G dibandingkan semester I pada 2019 merupakan langkah Indosat Ooredoo untuk melayani kebutuhan digital yang terus meningkat terutama saat era adaptasi kebiasaan baru akibat pandemi Covid-19.

    Selain itu, layanan pelanggan Indosat Ooredoo juga terus disiagakan untuk membantu kendala yang dihadapi saat ini. Pelanggan dapat menghubungi berbagai media sosial dan saluran digital secara Gratis untuk mendapatkan respon yang lebih cepat melalui Twitter, Instagram, Facebook, aplikasi MyIM3, website (www.indosatooredoo.com), email (cs@indosatooredoo.com), dan telpon ke *185#.
