Urutan Belajar Web Programming, Jagan Sampai Salah

By BakuraID Minggu, 30 Desember 2018 Comment Edit
Pendahuluan Web
Hallo pembaca setia bakuraid, kali ini bakura akan menjelaskan tentang Urutan Belajar Web Programming, Jagan Sampai Salah Urutanya artikel ini membahas tentang urutan bahasa pemrograman untuk membuat website secara sederhana. Sebelum ke pembahasan Anda harus tau dulu apa itu web programing.

Apa Itu Web Programing ?

Web programming adalah suatu proses, cara, atau pembuatan suatu program berbasis web. Dalam memahami web programming, yang harus Anda pelajari terlebih dahulu adalah pertama HTML, kedua CSS, ketiga PHP, ke empat Database, kelima JavaScript dan nanti akan ada tambahan. Langsung saja ke inti artikelnya simak pembahasanya.

Berikut merupakan Urutan Belajar Web Programming

1. HTML

Tahap awal dalam mempelajari web programming adalah menguasai dan memahami HTML (Hyper Text Markup Language) terlebih dahulu, Karena HTML adalah dasar dari halaman website. HTML merupakan suatu komponen dari pemrograman web yang cukup penting untuk dipelajari bagi para web developer. Sebab, secara garis besar HTML berfungsi untuk menampilkan tulisan, link, dan garis yang menjadi komponen utama pada suatu website.

2. CSS

Setelah HTML, yang harus dikuasai dan dipahami dalam mempelajari web programing adalah CSS (Cascading Styles Sheet). CSS (Cascading Style Sheet) adalah salah satu bahasa desain web (style sheet language) yang mengontrol format tampilan sebuah halaman web yang ditulis dengan menggunakan penanda (Markup Language). CSS berfungsi untuk mempercantik tampilan. Dengan CSS, kode HTML yang sudah dibuat dapat disempurnakan dan terlihat lebih menarik untuk dilihat. Jika Anda sudah menguasai HTML dan CSS, maka Anda sudah dapat membuat website statis dengan interface atau tampilan yang menarik.

3. PHP

Apabila Anda sudah menguasai HTML & CSS, maka selanjutnya yang Anda pelajari adalah PHP (Hypertext Processor). PHP adalah bahasa pemrograman script server-side yang didesain untuk pengembangan web. Bahasa ini banyak digunakan oleh para web developer karena mudah pelajari, dan tidak berbayar. Pada umumnya, PHP berfungsi untuk membuat website menjadi lebih dinamis dengan menyisipkan logika pemrograman pada sintaks HTML yang telah dibuat.
Namun, sebelum mempelajari PHP sebaiknya Anda mempelajari logika pemrograman terlebih dahulu. Ini akan membuat Anda dapat melakukan pemrograman secara lebih mudah.

3. MySQL (Database)

MySQL adalah singkatan “My Structured Query Language”. Program ini berjalan sebagai server menyediakan multi-user mengakses ke sejumlah database. MySQL adalah sebuah perangkat lunak sistem manajemen basis data SQL (bahasa Inggris: database management system) atau DBMS yang multialur, multipengguna. Database merupakan suatu komponen yang sangat penting dalam sistem informasi. Sebab, database merupakan wadah untuk menyimpan berbagai informasi yang akan di tampilkan kepada para pengunjung. Oleh karena itu, para web developer diharapkan mampu menganalisa kebutuhan program dan merancang database sehingga pembuatan program dapat berjalan dengan lebih cepat dan mudah.

4. JavaScript

Javascript adalah program dalam bentuk script, yang akan dijalankan oleh interpreter yang telah ditanamkan ke dalam browser web, sehingga browser web dapat memasukan program Javascript. Program Javascript dimasukan  ke dalam dokumen HTML dengan ditandai dengan tag yang diawali dengan <Script …> dan diakhiri dengan </script> sama seperti tag pada html (<>…</>). Javascript merupakan hal yang penting untuk dipelajari sebab antara PHP dan Javascript memiliki kekurangan dan kelebihan masing-masing. Dengan begitu, Anda dapat mengembangkan suatu situs dengan lebih mudah berdasarkan kekurangan dan kelebihan kedua bahasa pemrograman tersebut.

5. Tambahan (Library Dan Framework)

Urutan Belajar Web Programming yang terakhir yaitu tambahan untuk memper mudah anda dalam pembuatan website. Sebenarnya setelah Anda memahami dan menguasai bahasa pemrograman yang telah dijelaskan diatas, maka Anda sudah bisa disebut sebagai web developer. Akan tetapi, terdapat suatu tambahan yang sebaiknya Anda kuasai agar situs dapat dikembangkan dengan lebih menarik lagi yaitu library dan framework. apa itu library apa itu frame work kita akan bahas sekarang. Library adalah sekumpulan kode siap pakai untuk melakukan suatu tujuan tertentu. contoh dari library itu sendiri adalah Jquery, React, SweetAlert, dan lain lain. Dengan library ini, Anda dapat menghadirkan animasi dan efek-efek tertentu yang dapat membuat suatu situs terlihat lebih menarik. mungkin itu saja penjelasan singkat tentang library. sekarang kita bahas Framework. Framework adalah sebuah software untuk memudahkan para programer untuk membuat sebuah aplikasi web. Dengan framework, Anda dapat mengembangkan suatu situs dengan lebih mudah, cepat, aman, dan dinamis. Contoh framework PHP diantaranya adalah Codeigniter, Yii, Laravel dan lain sebagainya. Contoh framework CSS diantaranya adalah Bootstrap, Foundation, dan lain sebagainya.
